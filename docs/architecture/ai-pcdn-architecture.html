ai-pcdn 系统架构

节点接入、流量采集、95 值计费、监控告警与结算分析

三类接入

  • • 运营后台：节点、告警、账单、对账和利润大盘
  • • 个人门户：注册登录、自助上机、流量与账单查询
  • • 采集 Agent：分钟峰值、心跳、失败重试与 OTA 升级

核心业务

  • • 节点、流量、日滚动与月冻结 95 值
  • • 采购结算、销售对账、应收汇总与利润分析
  • • 告警收敛、恢复通知和 Agent 版本发布

部署与数据

  • • Gin、Vue 3 与 SQLite 支持本地或 Docker Compose 部署
  • • 定时执行离线检查、95 值、月账单与告警检查
  • • SQLite 和上传目录通过持久化卷保留
