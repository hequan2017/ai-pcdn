# 前端产品品牌统一为 ai-pcdn

- 状态：已完成
- 日期：2026-07-21

## 需求

将前端面向用户的 Gin-Vue-Admin 产品名称统一为 `ai-pcdn`。

## 合规边界

- 仓库 LICENSE 禁止在没有单独商业授权时删除、修改或遮蔽原版权、作者归属及授权提示。
- 用户已于 2026-07-21 明确确认取得单独商业授权，并要求继续删除版权页脚。
- 修改的上游文件增加了日期为 2026-07-21 的修改说明。

## 已实现

- 浏览器标题、导航标题、构建欢迎信息和项目设置页统一使用 `ai-pcdn`。
- 前端包名和主题导出文件名改为 `ai-pcdn`。
- 登录与封面 SVG 中的产品文字改为 `AI-PCDN`。
- 关于页面改为只展示 `ai-pcdn` 产品定位与技术栈，不再加载团队成员、仓库统计和提交记录。
- 通栏头部、纵向侧栏和移动菜单不再展示左上角 Logo，仅保留 `ai-pcdn` 产品名。
- 仪表盘页面保留路由组件但清空全部展示内容及相关数据逻辑。
- 后台布局与登录页不再渲染版权页脚，登录页关联的上游站点图标和无用样式同步移除。
- 构建日志、浏览器控制台、项目设置和 README 中的上游品牌及版权提示统一替换为 `ai-pcdn` 信息。

## 验证

- `npm run build` 构建成功。
- 构建日志显示包名与欢迎产品名均为 `ai-pcdn`。
